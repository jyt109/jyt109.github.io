Wanna not write 3 lines of code every time you loop through a list ??


There are few ways to do so, namely List Comprehenison, Map( ) and Lambda( ).


Here we are gonna cover when and how to use these functions ...


List comprehension


Writing a for loop in one line

In [1]: 
[i**2 for i in range(3)]
  Click to hide output, if there is one
Out[1]:
[0, 1, 4]


With conditions

In [5]: 
["odd" if i**2 % 2 == 1 else "even" for i in range(3) ]
  Click to hide output, if there is one
Out[5]:
['even', 'odd', 'even']
In [11]: 
["zero-sq" if i**2 == 0 else "one-sq" if i**2 == 1 else "two-sq" for i in range(3) ]
  Click to hide output, if there is one
Out[11]:
['zero-sq', 'one-sq', 'two-sq']
In [7]: 
["odd" for i in range(3) if i**2 % 2 == 1]
  Click to hide output, if there is one
Out[7]:
['odd']


More than 1 loop

In [15]: 
ll = [[1,2],[3,4]]

print [item + 1 for l in ll for item in l ]

print [item + 1 if item % 2 else item + 2 for l in ll for item in l ]

Map( )


Achieves the same result as list comprehension by mapping a function to every item in a list

Faster than list comprehension for built-in Python functions, such as str and int

Slower than list comprehension for self-defined functions


In [21]: 
test_list = list(range(30))
  Click to hide output, if there is one
In [48]: 
## MAP
%%timeit -n 1000 
map(str, test_list)
  Click to hide output, if there is one 
1000 loops, best of 3: 5.88 µs per loop
In [46]: 
## LIST COMPREHENSION
%%timeit -n 1000 
[str(i) for i in test_list]
  Click to hide output, if there is one 
1000 loops, best of 3: 11.6 µs per loop

Map( ) is faster here (Built-in function loops):

5.88 µs < 11.6 µs per loop


In [54]: 
## MAP
%%timeit -n 1000
map(lambda i: i**2, test_list)
  Click to hide output, if there is one 
1000 loops, best of 3: 9.5 µs per loop
In [55]: 
## LIST COMPREHENSION
%%timeit -n 1000 
[i**2 for i in test_list]
  Click to hide output, if there is one 
1000 loops, best of 3: 4.34 µs per loop

Map( ) is slower here (Self-defined function loops):

9.5 µs > 4.34 µs per loop


Lambda


A shortcut to define a function in one line. (As seen above)

We don't often use lambda with list objects, since we have list comprehension.


Lambda is very handy when it comes to looping through Pandas objects (DataFrame and Series).

The associated functions for looping through DataFrame and Series are apply( ), applymap( ), and map( ).

Map( )  For Pandas Series


Works same as map( ) for list.

Pandas Series has no list comprehension, so we use map( ).

Pandas Series has cool features that are not available for list, and is more suited for some data types, e.g. Time Series data.

Again, for simpler functions, Pandas Series allows things like "Series + 1" to add 1 to every item in the series.

In [112]: 
s = pd.Series([1,2,3,4])
print s.map(lambda x: x + 1)

print "\n\n"

print s + 1
